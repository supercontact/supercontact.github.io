Finding the shortest paths in 3D space

Mar. 2016

Ruoqi He, Chia-Man Hung

We established an algorithm capable of finding approximate shortest paths in 3D space with arbitrary obstacles. To achieve this, we first apply hierachical space division to build an octree representation of obstacles in space. We then convert it to a graph and apply Lazy Theta* algorithm to find an approximate shortest path in euclidien space constrained by line of sight checks. We also extend it to be able to find the shortest paths from multiple starting points at once, and achienved simutaneous navigation in 3D space for multiple agents without collision. All our results are shown in the demo.


Features include:

  • A demo containing different 3D scenes to navigate around, including a 3D maze
  • Add space ships and click on the map to set their destination
  • See planned shortest paths of space ships
  • Collision-free navigation
  • Path replanning when being pushed to the wrong way
  • Set different octree level to adjust approximate shortest path precision
  • Power testing with different graph type and path-finding algorithms

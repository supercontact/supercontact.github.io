Geodesics in Heat - Finding the shortest paths on 3D surfaces

Nov. 2015

Ruoqi He, Chia-Man Hung

We created an interactive demo that implements the heat geodesics algorithm (proposed by Crane 2013) which generates the approximate distance field (geodesics) from a given source on 3D surfaces. The algorithm uses heat propagation to approximate the direction of shortest paths. This algorithm only involves solving linear systems and can be used on real-time scenarios. We extended this algorithm to handle area sources or multiple sources instead of single point sources. We also implemented a navigation system that allows an agent to follow the shortest path to its destination on a 3D surface once the distance field is calculated.


Features include:

  • Click on the surface of a 3D model to set the source point and see the generated distance field (animated!)
  • Control-Click to set multiple sources, Shift-Click to set a continuous line source
  • See shortest paths to source from the mouse position by holding Alt
  • An additional small character will follow the shortest path to the source
  • 9 Different 3D models including ones with complex boundaries
  • Heat flow and gradient of distance field visualizations
  • Adjustable time step and boundary condition combine factors
